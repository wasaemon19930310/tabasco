# Tabasco Dodger (Unity 2D Action Game)
飛んでくるタバスコをジャンプで避ける、シンプルながらも中毒性のある2Dアクションゲームです。

## ゲーム概要

* **ルール**: 右側から次々と飛んでくるタバスコを避け続けます。
* **操作**: クリック（またはタップ）でプレイヤーがジャンプ。
* **ライフ制**: タバスコに当たるとHPゲージが減少し、ゼロになるとゲームオーバー（リトライ）となります。

## 技術スタック

* **Game Engine**: Unity (C#)
* **Programming Concepts**:
  * **オブジェクトの動的生成**: `Instantiate` を用いた敵オブジェクトの生成管理。
  * **物理演算**: `Rigidbody2D` を利用した自然なジャンプ挙動の実現。
  * **シングルトンライクな設計**: `GameDirector` によるゲーム状態（HP）の一元管理。
  * **トリガー判定**: `OnTriggerEnter2D` を用いた当たり判定処理。

## こだわりポイント
* **演出の工夫**: 単純に移動するだけでなく、`RotateAround` を使用してタバスコを回転させながら飛ばすことで、ゲームの視覚的な賑やかさを演出しました。
* **メモリ管理**: 画面外に消えたオブジェクトを `Destroy` で確実に破棄し、長時間プレイしても動作が重くならないよう配慮しました。
